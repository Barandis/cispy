DroppingBuffer

cispy/core/buffers~ DroppingBuffer

Source:
  • core/buffers.js, line 335

A buffer implementation that drops newly added items when the buffer is full.

This dropping behavior is silent: the new item is simply not added to the queue. Note taht this buffer is never full because it can always be added to wiehtout exceeding the size, even if that 'adding' doesn't result in a new item actually appearing in the buffer.
